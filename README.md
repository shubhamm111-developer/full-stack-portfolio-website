# Full Stack Portfolio

A personal portfolio built with the MERN stack — designed around an IoT/hardware "lab
telemetry" aesthetic: dark panel background, amber/cyan signal accents, sections framed as
hardware modules.

**🔗 Live demo:** _(add link after deployment)_

## ✨ Features

- 📡 Live project data served from MongoDB via a REST API (not hardcoded)
- 📬 Working contact form that persists messages to the database
- 📱 Fully responsive design
- 🧩 Clean component-based React architecture

## 🛠 Tech Stack

| Layer | Tech |
|---|---|
| Frontend | React (Vite), custom CSS design system |
| Backend | Node.js, Express |
| Database | MongoDB Atlas |

## 🚀 Running Locally

**Backend**

```bash
cd server
npm install
npm run seed

```

**Frontend**
```bash
cd client
npm install
npm run dev
```
